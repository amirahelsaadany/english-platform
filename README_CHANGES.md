# تحديث إدارة محتوى الموقع

## الملفات المعدلة والجديدة

### 1. app.py (مُحدَّث)
- **نموذج جديد**: `SiteContent` لتخزين المحتوى القابل للتعديل
- **دوال مساعدة**: `get_site_content()` و `get_all_site_content()`
- **مسار جديد**: `/teacher/site-content` لإدارة المحتوى
- **تحديث**: مسار `/` لتمرير `site_contents` للقالب

### 2. index.html (مُحدَّث)
- تم استبدال كل النصوص الثابتة بمحتوى ديناميكي من قاعدة البيانات
- الأقسام القابلة للتعديل:
  - قسم الهيدر الرئيسي (العنوان، الوصف، الأزرار)
  - قسم الاقتباس
  - قسم المميزات (6 ميزات)
  - عناوين أقسام الدورات والجلسات والإعلانات
  - قسم الفوتر

### 3. site_content.html (جديد)
- صفحة إدارة المحتوى في لوحة تحكم المعلم
- منظمة في تبويبات: الهيدر، المميزات، الأقسام، الاقتباس، الفوتر
- نموذج تحديث جماعي لحفظ كل التغييرات دفعة واحدة

### 4. قوالب المعلم (مُحدَّثة)
- `dashboard(1).html`
- `courses(1).html`
- `course_form.html`
- `course_edit.html`
- `live.html`
- `announcements.html`
- `students.html`
- `profile(1).html`
- `assistants(1).html`
- `add_assistant(1).html`

تم إضافة قسم "الإعدادات" في الشريط الجانبي مع رابط "محتوى الموقع"

## كيفية الاستخدام

1. سجّل دخولك كمعلم
2. اذهب إلى لوحة التحكم
3. اضغط على "محتوى الموقع" في القائمة الجانبية
4. عدّل أي نص تريده
5. اضغط "حفظ جميع التغييرات"
6. شاهد التغييرات مباشرة في الصفحة الرئيسية

## المفاتيح المتاحة للتعديل

| المفتاح | القسم | الوصف |
|---------|-------|-------|
| `hero_tag` | الهيدر | وسم الهيدر الصغير |
| `hero_title_en` | الهيدر | العنوان بالإنجليزية |
| `hero_title_ar` | الهيدر | العنوان بالعربية |
| `hero_subtitle` | الهيدر | وصف الهيدر |
| `hero_cta_primary` | الهيدر | زر الدعوة الرئيسي |
| `hero_cta_secondary` | الهيدر | زر الدعوة الثانوي |
| `feature_1_title` إلى `feature_6_title` | المميزات | عناوين الميزات الست |
| `feature_1_desc` إلى `feature_6_desc` | المميزات | وصف الميزات الست |
| `quote_text` | الاقتباس | نص الاقتباس |
| `quote_author` | الاقتباس | صاحب الاقتباس |
| `courses_title` | الأقسام | عنوان قسم الدورات |
| `courses_subtitle` | الأقسام | وصف قسم الدورات |
| `live_title` | الأقسام | عنوان قسم الجلسات |
| `live_subtitle` | الأقسام | وصف قسم الجلسات |
| `announcements_title` | الأقسام | عنوان قسم الإعلانات |
| `footer_brand_name` | الفوتر | اسم العلامة التجارية |
| `footer_brand_desc` | الفوتر | وصف العلامة |
| `footer_copyright` | الفوتر | نص حقوق النشر |
